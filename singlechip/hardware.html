上拉电阻的作用：
OC门要输出高电平 必须加上拉电阻
STC89C52内部上拉电阻为20K 最大输出电流为250uA 加上拉电阻可以增大驱动能力
在电平转换电路中 上拉电阻可以限流
单片机未使用的引脚 如总线引脚 引脚悬空时 容易受电磁干扰 增加功耗 加上/下拉电阻可以有效抵制电磁干扰
上拉电阻越大 驱动能力越强 但功耗越高 在开漏输出时 过大的上拉电阻会导致信号上升沿变缓 一般选用1~10K电阻
步进电机分反应式 永磁式 混合式
28BYJ48代表28mm 步进 永磁式 减速 4相8拍
此时14转子与C相绕组的夹角为11.25° 每次通电一个绕组 经过8个4节拍后定子转过一周 称单相绕组通电四节拍
在上述每两个节拍间插入一个节拍（双绕组通电）就会变为8个节拍 步进角度为5.625° 扭矩更大
除单四拍 八拍 还有双四拍（将单绕组通电换成双绕组通电）

步进电机的5根线 红 橙 黄 粉 蓝 对应 VCC A B C D
8拍工作模式通电顺序为A AB B BC C CD D DA A

若MC0对应P1^0 要导通A相 P1的低4位应为0b1110（0xE）
因此8拍控制代码为
unsigned char code BeatCode[] = {0xE, 0xC, 0xD, 0x9, 0xB, 0x3, 0x7, 0x6};
该电机的启动频率为>=550P.S.S(每秒脉冲数） 即间隔应大于1/550s（1.8ms）
