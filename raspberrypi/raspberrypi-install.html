RaspberryPi 初始化
  1. 下载Raspberry OS.zip
  2. 使用Win32DiskImager.exe将Raspberry.img刻录到TF卡
  3. 开启ssh vnc
  4. 更改password为wuchi57spi
  5. 用户默认为pi hostname默认为raspberrypi
  6. 将树莓派连接网线
  7. 使用windows cmd 连接树莓派 ssh pi@raspberrypi
  8. 使用sudo passwd root 创建Root 并设置密码wuchi57spi 可以使用su root切换用户
  9. 更换源：https://mirror.tuna.tsinghua.edu.cn/help/raspbian/
  10. 安装vim：sudo apt-get install vim
    更改配色：sudo vim /etc/vim/vimrc 添加语句 colorscheme ron
    其他配色位于：/usr/share/vim/vim80/colors/
  11. 可以在windows使用VNC Viewer连接树莓派 通过ifconfig得到ip地址 输入用户名和密码即可连接树莓派 可以继续完成之前没完成树莓派配置向导
  12. 移除首次安装配置向导
    sudo rm /etc/xdg/autostart/piwiz.desktop
    sudo reboot now
  13. 根据lcdwiki配置分辨率
