摄像头配置


1.首先使用 ls -al /etc 指令来查看是否加载到了对应的video device设备： 如果没有发现我们的设备，接下来要做的是添加摄像头的驱动程序.ko文件 sudo vim /etc/modules 在最后添加如下的代码：bcm2835-v4l2 和对应的raspiberry B3+的硬件使能问题 sudo raspi-config 开启Camera 并重启 2.使用内置软件 raspistill -o image.jpg raspivid -o mykeychain.h264 -t 10000 -w 1280 -h 720

使用raspi-config配置树莓派




使用df -h 查看树莓派硬盘空间使用情况




ARP（Address Resolution Protocol）即地址解析协议 用于实现从 IP 地址到 MAC 地址的映射，即询问目标IP对应的MAC地址。


如在树莓派输入：arp -a 第一个为路由器地址 第二个为电脑地址



输入 arp -a 弹出的IP 地址和MAC地址，注意找开头是b8 的（树莓派一般就b8）


如在电脑输入：arp -a
