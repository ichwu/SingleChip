unsigned int I = 10;
int n = -42
n + n = -84
I + n = 4294967264


如42 为字面常量 int为最小十进制字面常量 即short非字面常量类型
-42为字面常量int的42的取反值
默认浮点数的字面常量为double
还有字符和字符串字面常量
字符串即为字符数组 字符串的字面值的实际长度比内容多1 即'\0' 

转义字符：\t  \"  \'  \? \\ \r  \n
\后也可接八进制和十六进制数字
使用前缀和后缀可以改变整型 浮点型 字符型的默认字面常量类型 如
前缀
u      char16_t     Unicode 16 字符
U      char32_t    Unicode 32 字符
L       wchar_t     宽字符
u8     char            UTF-8(仅用于字符串字面常量）
后缀
u or U    unsigned 
l or L        long
ll or LL      long long
浮点后缀
f or F          float
l or L           double

对于一个整型字面常量 我们可以单独指定其是否带符号和占用空间多少
true和false是bool的字面常量
nullptr是指针的字面常量
