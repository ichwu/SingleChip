C 关键字



C的精神：信任程序员
		  不要妨碍程序员做需要做的事
		  保持语言精练简单
		  只提供一种方法执行一项操作
		  让程序跑得更快，即使不能保证其可移植性
		（针对目标计算机定义最合适的某种操作，而不是
			强加一个抽象 统一的定义）

C的标准：K&R C
		   ANSI C | ISO C | ANSI/ISO C | C89/90
		   C99
		   C11  

使用C的7个步骤：定义程序目标
				  设计程序
				  编写代码
				  编译代码
				  运行程序
				  测试和调试代码
				  维护和修改代码
